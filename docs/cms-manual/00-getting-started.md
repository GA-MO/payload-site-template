# Getting Started

Welcome — this is the content editor's quick tour of the admin panel.

## Logging in

Go to `/admin`. Use the email + password you registered on the first run.

## What's where

Each collection in the left nav holds one type of content (e.g. articles, products, team members). Globals (if any) are one-off pages or settings — there's only one of each.

## Editing content

1. Click a collection in the left nav.
2. Click "Create New" or pick an existing item.
3. Fill in the fields. Required fields are marked.
4. Click **Save** — your change is live on the public site within a minute (or instantly, depending on caching).

## Media

Upload images via the **Media** collection. Add a meaningful **Alt text** for accessibility. The system automatically generates multiple sizes (thumbnail, card, feature).

## Need help?

Browse the other docs in this manual (left nav) or use the search bar (⌘K).
